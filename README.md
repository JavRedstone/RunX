# RunX

A better and up-to-date version of the game Run 3

# How to play

Go to [runx.pages.dev/](https://runx.pages.dev/)

# Features

- Smooth gameplay
- 3D graphics
- Three JS
- 6 Tile types
- 2 Game modes
